Welcome

<<<<<<< HEAD

This book is about the analysis of household surveys from South Asian countries and is intended to be used as a manual by any researcher interested in using the South Asia Regional Micro Database (SARMD). The household surveys contained in SARMD provide a rich and open source of data on economic behavior and expenditure patterns, demographics, educational attainment, employment, the acquisition of durable assets, and housing. The harmonization of these surveys allows to compare social and economic statistics among the eight countries in the region.

=======

This book will teach you everything you need to know about the South Asia Region MicroDatabase (SARMD). You will learn how to get acces to the microdata through the datalibweb system, how to perform basic (and not so basic) socioeconomic calculations, howe comparable and similar household surveys are across the region,how good the harmonization process is, and how to make analysis at the regional or country level by providing a series of case studies and findings.

This book has been written in Rmarkdown syntax and compiled with the Bookdown package. The source files are available in the Word Bank Github repo DDDD. In order to become a contributor of this project, please send an email to … requesting for access. Once you’re granted for access, you may clone of fork this repo and start contributing.

  2. Everything is written using plain text, so you may enjoy the benefits of tracking changes and collaborative platforms as Git.
  3. You can execute R or Stata code directly from the text as well as include code chunks and/or its corresponding results inline wherever you want within the document.
  4. You can add features like multi-page HTML output, numbering and cross-referencing figures/tables/sections/equations, inserting parts/appendices, and imported GitBook style to create elegant and beautiful books.
